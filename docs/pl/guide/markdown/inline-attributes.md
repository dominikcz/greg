# Atrybuty inline dla linków i obrazów

Możesz dodać atrybuty HTML do linków i obrazów przez blok `{...}` bezpośrednio po `)`:

```md
[Otwórz w nowej karcie](https://example.com){target="_blank" rel="noopener"}

[link stylowany](../routing){class="my-link"}
```

Output:

[Otwórz w nowej karcie](https://example.com){target="_blank" rel="noopener"}

[link stylowany](../routing){class="my-link"}
